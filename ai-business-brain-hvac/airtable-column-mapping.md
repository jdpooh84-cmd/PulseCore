# Airtable Column Mapping (Copy Exactly)

Use these exact Airtable column names in your **Leads** table:

| Airtable Column Name | Type in Airtable | Comes From |
|---|---|---|
| Lead ID | Single line text | Auto-generated by webhook |
| Created At | Date and time | Auto-generated by webhook |
| Report Month | Single line text | Auto-generated by webhook (YYYY-MM) |
| Business Name | Single line text | Payload `business_name` (default: Coastal Comfort HVAC) |
| Business City | Single line text | Payload `business_city` (default: Chesapeake) |
| Business State | Single line text | Payload `business_state` (default: Virginia) |
| Offer Name | Single line text | Payload `offer_name` (default: AI Business Brain) |
| Service Category | Single line text | Payload `service_category` (default: HVAC) |
| First Name | Single line text | Payload `first_name` |
| Last Name | Single line text | Payload `last_name` |
| Full Name | Single line text | Webhook combines first + last |
| Email | Email | Payload `email` |
| Phone | Phone number | Payload `phone` |
| Service Needed | Single line text | Payload `service_needed` |
| Urgency | Single select (Emergency, Same-Day, This Week, Normal) | Payload `urgency` (default: Normal) |
| Message | Long text | Payload `message` |
| Source Page | Single line text | Payload `source_page` |
| UTM Source | Single line text | Payload `utm_source` |
| UTM Medium | Single line text | Payload `utm_medium` |
| UTM Campaign | Single line text | Payload `utm_campaign` |
| Status | Single select (New, Contacted, Won, Lost) | Webhook default: New |
| Owner Alert Sent | Single select (Yes, No) | Webhook updates after owner email |
| Customer Auto-Reply Sent | Single select (Yes, No) | Webhook updates after customer email |
| Owner Alert Sent At | Date and time | Webhook updates after owner email |
| Customer Auto-Reply Sent At | Date and time | Webhook updates after customer email |

## Important
- If column names do not match exactly, Airtable API writes will fail.
- Keep spelling, spaces, and capitalization exactly the same.
